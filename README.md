# Invoice Generator (Windows)

This repository is for **release downloads only**.

## Download

1. Open the latest release:
   [Releases](https://github.com/ChowdhuryMainak/Invoice_Generator/releases/latest)
2. Download `InvoiceGenerator.zip`
3. Extract the zip
4. Run `Invoice Generator.exe`

## Notes

- Works offline with a local database
- Windows desktop application (`.exe`)
